# crypto-trend

twitterAPI を用いた仮想通貨情報収集ツール。

## 簡単な説明

twitter 上の仮想通貨に関するつぶやきを銘柄ごとに自動で集計し、銘柄の話題性をランキングを知ることができます。
また、twitter アカウントを連携することで、仮想通貨に関連するユーザーを自動でフォローすることが可能です。

## サービスの主要機能

-   銘柄の話題性ランキング表示(１時間、1 日、１週間ごと)
-   仮想通貨関連アカウント表示
-   アカウントのフォロー・アンフォロー
-   アカウントの自動フォロー

## その他の機能

-   ログイン・ログアウト
-   会員登録
-   退会
-   SNS 連携(twitter)
-   プロフィール編集
-   パスワードリセット
-   パスワード変更
